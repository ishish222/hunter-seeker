one global file (global.conf), containing:
registered generators with ids
registered fuzzboxes with ids

one file per fuzzbox, named by id, containing:
generator id
parameters for instrumentation (app path)
special instrumentation extensions
step value (after how many tests should fuzzbox report?)
gen_step value (after how many tests should generator provide more samples?)
gen_amt value (how many samples should the generator provide?)
